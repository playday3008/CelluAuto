# CelluAuto

CelluAuto is a demo project for creating cellular automata simulations in JavaScipt.

## Libraries used

- [PixiJS](https://www.pixijs.com/) for rendering
- [Tweakpane](https://tweakpane.github.io/docs/) for UI
